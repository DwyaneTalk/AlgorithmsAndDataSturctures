# dataStructure

basic dataStrucure realization using c/c++ 

### including:
* **线性结构**
    * 线性顺序表：SqList    
    * 线性链式表：LinkList  
    * 队列：Queue   
    * 栈： Stack
    * 堆： Heap
    * 哈希：Hash
* **树形结构**  
    * 二叉树：BinaryTree
    * 二叉线索树：BiThrTree
    * B-树：BMinusTree
    * Trie树（字典树）：TrieTree
* **网状结构**  
    * 无向图：UDGraph
    * 有向图：DGraph
* **查找排序**
    * 静态查找表：StaticSrhTable
    * 动态查找表：DynamicSrhTable
    * 通用排序库：Sort
